Comenzi necesare pentru rulare:

Backend:

In terminal, la nivel "root" (>> cd live-chat):

>> npm init 
>> npm install 
>> npm install cors express nodemon socket.io (pentru instalare backend) 
>> npm install -g \@angular/cli (pentru instalare frontend) 
>> ng new live-chat-frontend
>> npm start (pentru rulare backend, port 3000)

Frontend:

In terminal, la nivelul folder-ului "live-chat-frontend":

>> cd live-chat-frontend 
>> npm install 
>> npm install socket.io-client 
>> ng serve (pentru rulare frontend, port 4200)
